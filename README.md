# Hex Board

Hex Board is an addon for Hex Casting (Minecraft 1.20.1) that introduces an in-world board workflow for composing and editing hex patterns.

In short, this mod adds:

- A connectable `Board` block system for slot-based pattern organization.
- Multiple `Type Block` items (Quartz, Wool, Lapis, Redstone) for storing different iota data.
- A `Staff For Board` with dedicated edit modes (add, delete, move, copy, resolve, literal resolve, load).
- Five board-operation patterns (`load`, `delete`, `move`, `resolve`, `literal_resolve`) for direct spell-side board manipulation.
- Patchouli integration for in-book navigation and pattern-assisted placement.
